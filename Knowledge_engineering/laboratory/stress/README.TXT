Stress 1.20


Hartelijk dank voor uw bijdrage in de ontwikkeling van Stress. Dit is de
Windows 3.1 versie van Stress. Deze diskette bevat de volgende files:

        -README.TXT       deze text;
        -STRESS.EXE       het programma;
        -EDITOR.EXE       de Stress editor;
        -AGENDA.EXE       de Stress Agenda;
        -BLACKBOA.EXE     de Stress Blackboard;
        -TRACE.EXE        de Stress Trace faciliteit;
        -STRESS.HLP       de WinHelp file voor Stress;
        -EDITOR.HLP       de WinHelp file voor de Editor;

        -GAME_BKE.KBS     voorbeelden.
        -OSCILLA1.KBS
        -OSCILLA2.KBS
        -TRAVEL1.KBS
        -TRAVEL2.KBS
        -TRAVEL3.KBS
        -WERKLOOS.KBS

NOTA BENE: Deze programma's mogen niet worden verspreid zonder uitdrukkelijke
           toestemming van de auteur. Eventuele schade ontstaan door het
           gebruik van deze software valt volledig onder de verantwoordelijkheid
           van de gebruiker en kan niet op de auteur worden verhaald.

Stress kan op de volgende manier geinstalleerd worden:

        1. Maak een directory (bijvoorbeeld c:\STRESS).
        2. Copieer alle files van de disk naar deze directory.
        3. Installeer de exe-files van Stress in de Program Manager
           door New te kiezen in het File menu.
